### 开发模式

- 安装依赖库

```
npm install
```

- 启动开发模式

```
npm run dev
```

- 浏览器访问

[http://localhost:9000/](http://localhost:9000/)

如需更改端口号，请在 `scripts/dev.js` devServer 配置项中修改 port。

- 打包输出

```
npm run build
```
